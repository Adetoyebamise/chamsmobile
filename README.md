## Chamsmobile Backend 

### Requirements

 - [Node v7.6+](https://nodejs.org/en/download/current/)
 - [NPM](https://yarnpkg.com/en/docs/install)

### Getting Started

Clone the repo:

```bash
git clone https://github.com/adetoyebamise/credit-app.git
cd credit-app
```
Install npm:

```bash
npm install
```

Install dependencies:

```bash
npm install ...
```

### Running Locally

```bash
npm run start:dev
```
